◆ load_ini()

void std::ConfigParser::load_ini ( string  filename)
inline

Parses a config file with basic INI structure lines starting with ; and # are comments, category or sections are bracketed [section] when accessing them by the get_string or get int function variables under section heads [title] are accessed by title.variablename See test cases for basic usage This function will throw an exception if the file doesn't exist, or the file is writeable by the executor This enforced /etc/ security best practice.

Parameters
filenameof file to be loaded for parameters

Don't try and open the file, and throw an exception, if the file does not exist or is not a regular file.

We don't want the user executing the executable to be able to write to the ini by default

if the file is not open abort
